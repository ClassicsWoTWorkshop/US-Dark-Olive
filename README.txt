Thank you for downloading!

To install:
- Drag the "gui" and "vehicles" folders from inside the MAIN MOD - COLOR CHANGE folder to <Your WoT Install Directory>\res_mods\<version> (EXAMPLE: C:\Games\World_of_Tanks\res_mods\0.9.20.0).
Optional:
- If you want to also remove the skin from the T26E5 Patriot or Chrysler K GF, drag the "gui" and "vehicles" folders from inside the "ADDON - Remove T26E5 Patriot skin" or "ADDON - Remove Chrysler K GF skin (original by Avalon304)" folder to your World of Tanks install directory as well. 

Enjoy!
